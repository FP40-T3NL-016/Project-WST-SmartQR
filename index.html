Browser-Based QR Project

Smart QR Code Reader and Data Analyzer

This web application reads QR code data through camera, uploaded QR image and manual input. It analyzes embedded information using JavaScript pattern recognition and classifies URLs, phone numbers, emails, WiFi records, product codes, payment data and plain text.

The interface uses a responsive flex layout, external CSS, branded images, dark mode, localStorage history and clean analytics tables.
